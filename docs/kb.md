# KB、Kb、M之间的区别
---

计算机中信息都是二进制的`0`和`1`，其中每个`0`或`1`都被称作一个`位`。  
* `b`，即`bit(位)`  
* `B`，即`byte(字节)`  
* 一个字节 = 八个位，即`1B = 8b`  
* `K`，表示千的意思，一般文件的大小都使用`字节(KB)`来表示  
* 1KB = 8Kb  

## Kb和Kbps区别
* `ps`指的是'/s',即每秒  
* `Kbps`指的是网络速度，也就是每秒传送多少千个位的信息，也记作`Kb/s`
* `KBps`则表示每秒传送多少千字节  
* 同样有`1KBps` = `8Kbps`  

## M与Kbps、KBps之间的关系
为了在直观上显得网络的传输速度较快，一般公司都是用`kbps`千字节来表示带宽。   
* 一般宽带所说的多少兆，也就是`Mbps`
* `1M` = `1024Kbps` = `128KBps`
